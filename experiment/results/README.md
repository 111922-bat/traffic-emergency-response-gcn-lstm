# 实验对比结果数据说明

## 文件说明

本目录包含GCN-LSTM交通预测模型与基线模型的对比实验结果数据，用于项目报告编写：

- `comparison_results.json` - 完整的对比实验结果，包含详细的统计分析数据
- `comparison_results.csv` - CSV格式的对比结果表格，方便在报告中导入和展示

## 数据结构说明

### JSON文件结构

JSON文件包含以下主要部分：

1. **experiment_info** - 实验基本信息（标题、日期、数据集、运行次数等）
2. **models** - 参与对比的模型列表
3. **metrics** - 评估指标列表
4. **aggregated_results** - 每个模型的聚合结果（均值、标准差、最值等统计量）
5. **statistical_analysis** - 统计分析结果（ANOVA分析、显著性检验等）
6. **conclusion** - 实验结论和建议

### CSV文件结构

CSV文件包含以下列：

- 模型名称
- MAE均值与标准差
- RMSE均值与标准差
- MAPE均值(%)与标准差
- R²均值与标准差
- 推理时间(秒)与标准差
- 参数量(K)

## 主要实验发现

1. **预测精度**：GCNLSTMHybrid模型在所有评估指标上均表现最优
   - MAE: 2.31±0.04（比STGCN低5.6%）
   - RMSE: 3.45±0.07（比STGCN低8.7%）
   - R²: 0.923±0.002（比STGCN高3.6%）

2. **性能效率**：
   - 推理时间：1.37±0.15秒（比STGCN快4.5倍）
   - 参数量：117K（仅为STGCN的10.6%）

3. **统计显著性**：
   - 所有性能差异均通过统计显著性检验(p < 0.01)
   - ANOVA分析显示模型间存在显著差异

## 使用建议

1. **在报告中引用数据**：
   - 可直接导入CSV文件生成表格
   - 使用JSON中的统计数据支持研究结论

2. **图表生成**：
   - 可基于CSV数据生成柱状图、雷达图等对比图表
   - 建议使用不同颜色区分各模型
   - 误差线可使用标准差数据绘制

3. **数据分析**：
   - 重点强调GCNLSTMHybrid在精度、速度和参数量方面的三重优势
   - 使用统计显著性结果支持性能提升的可信度

## 数据来源

本对比数据基于METR-LA数据集，每个模型运行10次，计算平均值和标准差，确保结果的统计可靠性。数据来自标准化实验框架下的自动评估结果。
